## ğŸš€ How to Use

### Receipt Printing
1. **Add items to cart** and proceed to checkout
2. **Complete payment** by entering amount and selecting payment method
3. **Receipt Modal appears** showing:
   - Transaction ID
   - All items with prices
   - Subtotal, Tax, Total
   - Payment method
4. **Click "Print Receipt"** to print or **"Done"** to skip

### Barcode Scanner
1. **Connect USB barcode scanner** to your computer
2. **Click the barcode input field** at top of products section
3. **Scan the product barcode** or type it manually
4. **Press ENTER** to add product to cart
5. **If product not found:**
   - Error modal shows the barcode
   - Click "Try Again" to retry
   - Or manually search for product

### Manual Product Search
1. Use **"Search products..."** field
2. Type product name or code
3. Click product from results to add to cart

## âš ï¸ Troubleshooting

### Barcode Scanner Not Working
1. Physically connect scanner to USB port
2. Refresh the POS page (F5)
3. Click on barcode input field
4. Try scanning again

### Modals Not Appearing
1. Clear browser cache (Ctrl+Shift+Del / Cmd+Shift+Del)
2. Refresh page (F5)
3. Try different browser
4. Check browser console (F12) for errors

### Product Not Found
1. Verify product exists in database (Manage Products)
2. Check barcode is entered correctly
3. Use manual search instead
4. Add product if missing
